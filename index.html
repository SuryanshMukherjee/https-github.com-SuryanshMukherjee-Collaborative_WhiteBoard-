Learn: How This Project Was Built

This collaborative whiteboard was developed using HTML Canvas for drawing, JavaScript for user interaction, Socket.IO for real-time communication, and Node.js with Express as the backend server.

When you draw on the canvas, your mouse movements and drawing data are sent to the backend server via Socket.IO. The server broadcasts this data to all connected clients, ensuring every user sees updates in real time.

Help: How to Use This Project

  • Click and drag on the canvas to draw.
  • Use the Color picker to select your preferred drawing color.
  • Adjust the Size slider to control brush thickness.
  • Click Clear to erase the board for everyone.
  • Open the app in multiple browsers or devices to collaborate in real-time.
  • Ensure the Node.js server is running on localhost:8080.
